# Kovers

Kovers — это оригинальный интерфейс приватного мессенджера, вдохновлённый идеями канального общения и личных чатов.

## Почему это безопаснее по авторским правам

- Использован собственный бренд **Kovers**.
- Нет логотипов, названий, иконок и медиа сторонних сервисов.
- Макет и стили сделаны с нуля.

## Запуск на ПК

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 4173
```

Затем перейдите на `http://localhost:4173`.

## Запуск через Termux (Android)

1. Установите зависимости:

```bash
pkg update -y && pkg upgrade -y
pkg install -y git python
```

2. Склонируйте проект и перейдите в папку:

```bash
git clone <URL_ВАШЕГО_РЕПО>
cd Kovers
```

3. Запустите локальный сервер:

```bash
python -m http.server 4173
```

4. Откройте в браузере телефона:

- `http://127.0.0.1:4173` (если открываете на этом же телефоне), или
- `http://localhost:4173`

### Полезно

- Остановить сервер: `Ctrl + C`.
- Если порт занят, запустите на другом порту, например: `python -m http.server 8080`.
