# Kovers

Полностью рабочий чат-сервис: регистрация, вход, групповые и личные чаты, сообщения, голосовые/видео звонки (сигнализация), приватные комнаты.

## Что реализовано сейчас

- Регистрация и вход (`/api/auth/register`, `/api/auth/login`).
- Пустой аккаунт при первом входе: **нет фейковых чатов и сообщений**.
- Создание групповых чатов и личных диалогов (DM).
- Отправка/получение сообщений в реальном времени через polling.
- Голосовые и видео-звонки (создание, вход, завершение звонка).
- 500 отдельных файлов настроек чатов в `settings/chat/*.json`.

## Запуск

```bash
node server.js
```

Открыть: `http://localhost:4173`.

## Структура

- `server.js` — backend + API.
- `index.html`, `script.js`, `styles.css` — клиентское приложение.
- `kovers-data.json` — база данных (JSON).
- `settings/chat/` — 500 файлов конфигурации настроек.

## Важно

Клиентский код (HTML/CSS/JS) в браузере полностью скрыть нельзя. Для приватности держите репозиторий приватным и выносите критичную логику на backend.
