# Kovers

Kovers — это оригинальный интерфейс приватного мессенджера, вдохновлённый идеями канального общения и личных чатов.

## Как скачать (быстро)

- **С GitHub:** `Code` → `Download ZIP`.
- **Через Termux:**

```bash
pkg install -y git
git clone <URL_ВАШЕГО_РЕПО>
```

## Почему это безопаснее по авторским правам

- Использован собственный бренд **Kovers**.
- Нет логотипов, названий, иконок и медиа сторонних сервисов.
- Макет и стили сделаны с нуля.

## Запуск на ПК

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 4173
```

Затем перейдите на `http://localhost:4173`.

## Запуск через Termux (Android)

1. Установите зависимости:

```bash
pkg update -y && pkg upgrade -y
pkg install -y git python
```

2. Склонируйте проект и перейдите в папку:

```bash
git clone <URL_ВАШЕГО_РЕПО>
cd Kovers
```

3. Запустите локальный сервер:

```bash
python -m http.server 4173
```

4. Откройте в браузере телефона:

- `http://127.0.0.1:4173` (если открываете на этом же телефоне), или
- `http://localhost:4173`

## Как скачать всё в ZIP

### Вариант 1: через GitHub (проще)

1. Откройте страницу репозитория на GitHub.
2. Нажмите **Code** → **Download ZIP**.
3. Получите архив проекта целиком.

### Вариант 2: через Termux/ПК командой

Внутри папки проекта выполните:

```bash
zip -r Kovers.zip . -x ".git/*"
```

Если `zip` не установлен (в Termux):

```bash
pkg install -y zip
```

## Как скрыть код от других

Полностью скрыть код можно только организационно (настройкой доступа), а не в браузере.

1. Сделайте репозиторий **Private** на GitHub/GitLab.
2. Не отправляйте исходники публично, а делитесь только:
   - приватной ссылкой приглашённым людям,
   - или собранной версией/архивом без `.git`.
3. Если размещаете сайт в интернете, помните:
   - HTML/CSS/JS в браузере пользователь всегда может посмотреть.
   - Для «скрытия логики» переносите важную часть на сервер (backend/API), а не в фронтенд.

### Полезно

- Остановить сервер: `Ctrl + C`.
- Если порт занят, запустите на другом порту, например: `python -m http.server 8080`.
