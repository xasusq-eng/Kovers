# Kovers

Kovers — полноценный рабочий MVP приватного чата: авторизация по нику, комнаты, отправка и получение сообщений, сохранение истории на backend.

## Что уже работает по‑настоящему

- Backend на Node.js (`server.js`) без внешних зависимостей.
- Реальные API-эндпоинты (`/api/auth/guest`, `/api/rooms`, `/api/messages`).
- Многокомнатный чат с хранением истории в `kovers-data.json`.
- Живое обновление сообщений через polling (каждые ~1.2 секунды).

## Быстрый запуск

```bash
node server.js
```

Откройте: `http://localhost:4173`.

## Запуск через Termux (Android)

```bash
pkg update -y && pkg upgrade -y
pkg install -y git nodejs
git clone <URL_ВАШЕГО_РЕПО>
cd Kovers
node server.js
```

После запуска откройте в браузере телефона: `http://127.0.0.1:4173`.

## Важно про «скрыть код»

- Если проект отдаёт HTML/CSS/JS в браузер, этот клиентский код скрыть полностью нельзя.
- Чтобы ограничить доступ к исходникам: храните репозиторий приватным и не публикуйте исходный код.
- Критичную логику и приватные ключи всегда держите только на backend.
